# Multi-race TARGET HTML input

Use the same bundle ID as `tools/csv/input/races/`.

```text
tools/target-html/input/races/<bundle-id>/
  training-slope.html
  training-wood.html
  pedigree/
    <horse-name>.html
```

HTML remains optional input. Missing HTML must not block current-race or odds validation, and raw HTML must not be committed.
